# Wipro_Capstone_Project
Selenium WebDriver Capstone Project | Wipro SDET Training. Automates 27 GUI elements — forms, dropdowns, date pickers, file uploads, alerts, drag-and-drop, slider, and dynamic buttons using Java, Maven, Actions class, and JavascriptExecutor.
Selenium WebDriver Capstone Project | SDET Training @ Wipro
This project is a comprehensive Selenium WebDriver automation script developed as the Capstone Project for the SDET (Software Development Engineer in Test) training program at Wipro. The script automates all interactive GUI elements on the test practice website — testautomationpractice.blogspot.com — using Java and Maven.
What's Automated:
The script covers end-to-end interaction with every major web element type including text fields (Name, Email, Phone, Address), radio buttons, checkboxes, single and multi-select dropdowns, three types of date pickers, single and multiple file uploads, and a Wikipedia search widget with keyboard interaction.
Advanced Interactions:
Using Selenium's Actions class, the script performs double-click, mouse hover, drag-and-drop, and slider movement. All three alert types — Simple, Confirm, and Prompt — are handled programmatically. A dynamic START/STOP toggle button is automated using the same WebElement reference. JavascriptExecutor is used for setting read-only date fields and full-page scroll up and down operations.
Tech Stack:

Language: Java
Framework: Selenium WebDriver 4.43.0
Build Tool: Apache Maven
Browser Management: WebDriverManager 6.3.3
IDE: Eclipse

Key Highlights:

27 GUI interactions automated in a single script
Detailed console logging for every action
Clean, well-commented code with proper structure
No external test framework dependency — runs as a plain Java main class
